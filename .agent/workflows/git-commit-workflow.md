### 💡 한국어 커밋 메시지 작성 팁

이 룰을 적용하면 AI가 다음과 같은 스타일로 메시지를 뽑아줍니다:

* **제목:** `[Feat](Payment): 결제 검증 로직 추가` (명사형 종결)
* **본문:**
    * `결제 금액 위변조 방지를 위해 검증 단계 강화`
    * `PaymentService 내 중복 코드 리팩토링`

이제 `git add .` 하신 뒤 AI에게 **"커밋 메시지 작성해줘"**라고만 하시면 됩니다.
